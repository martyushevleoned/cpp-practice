# vim

## Normal mode
### Навигация
* `h` - влево
* `j` - вниз
* `k` - вверх
* `l` - вправо
* `w` - на слово вперед
* `b` - на слово назад
* `0` - в начало строки
* `$` - в конец строки
### Редактирование текста
* `u` - отменить
* `Ctrl+r` - переделать
* `x` - удалить символ
* `dw` - удалить слово
* `dd` - удалить строку

## Insert mode
* `i` - включить Insert mode
* `Ctrl+[` / `Esc` - вернуться в Normal mode

## Command-line mode
* `:` - включить Command-line mode
* `:!<command>` - выполнить команду

### Управление окнами
* `:q` - выйти
* `:q!` - выйти без сохранения
* `:wq` - записать изменения и выйти
* `:ex <path>` - открыть файл
* `:Hex <path>` - открыть с горизонтальным разделением
* `:Vex <path>` - открыть с вертикальным разделением
* `Ctrl+w` + `h/j/k/l` - навигация по окнам
### Терминал
* `:term` + `i` - открыть терминал в Insert mode
* `Ctrl+\` + `Ctrl+n` - вернуться в Normal mode

## Visual mode
* `v` - включить Visual mode
* `Ctrl+[` / `Esc` - вернуться в Normal mode
